# strengthTracker

## Description
strengthTracker est une application permettant le suivi des performances des sportifs. Elle offre des fonctionnalités pour enregistrer, visualiser et analyser les progrès dans divers exercices et activités physiques.

## Stack Utilisée
- ![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB) **React** : Une bibliothèque JavaScript pour construire des interfaces utilisateur.
- ![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white) **TypeScript** : Un sur-ensemble de JavaScript qui ajoute des types statiques.
- ![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white) **Vite** : Un build tool qui améliore la vitesse de développement.
- ![ESLint](https://img.shields.io/badge/ESLint-4B32C3?style=for-the-badge&logo=eslint&logoColor=white) **ESLint** : Un linter pour identifier et signaler les motifs dans le code JavaScript.

## Objectif de l'Application
strengthTracker a pour but de proposer un suivi complet et personnalisé des performances des sportifs. Les utilisateurs peuvent :
- Enregistrer leurs séances d'entraînement.
- Suivre leurs progrès dans divers exercices.
- Visualiser des graphiques de performance.
- Recevoir des analyses et des conseils personnalisés pour améliorer leurs performances.

## Installation et Utilisation
### Prérequis
- Node.js
- npm ou yarn

### Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Al-vallon/strengthTracker.git
   ```
2. Accédez au dossier du projet :
   ```bash
   cd strengthTracker
   ```
3. Installez les dépendances :
   ```bash
   npm install
   ```
   ou
   ```bash
   yarn install
   ```

### Lancer l'Application
Pour démarrer l'application en mode développement :
```bash
npm run dev
```
ou
```bash
yarn dev
```

### Tester le Projet
Pour exécuter les tests :
```bash
npm test
```
ou
```bash
yarn test
```
